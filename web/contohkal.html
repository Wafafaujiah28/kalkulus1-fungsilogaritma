Contoh Soal Fungsi Logaritma

1. Diberikan fungsi f(x) = log2 (x). Tentukan titik koordinat dari f(x) ketika x = 8.

Untuk menemukan titik koordinat, kita perlu menghitung nilai fungsi f(x) pada x = 8.
Langkah pertama adalah menghitung f(8) = log2 (8).
Kita tahu bahwa log2 (8) = 3 karena 23 = 8.
Oleh karena itu, titik koordinat yang dicari adalah (8, 3).
Jawaban: Titik koordinat adalah (8, 3)

2. Diberikan fungsi f(x) = log10 (x). Tentukan titik koordinat dari f(x) ketika x = 100.

Untuk menemukan titik koordinat, kita perlu menghitung nilai fungsi f(x) pada x = 100.
Langkah pertama adalah menghitung f(100) = log10 (100).
Kita tahu bahwa log10 (100) = 2 karena 102 = 100.
Oleh karena itu, titik koordinat yang dicari adalah (100, 2).
Jawaban: Titik koordinat adalah (100, 2)

3. Diberikan fungsi f(x) = loge (x). Tentukan titik koordinat dari f(x) ketika x = 1, x = e, dan x = e2.

Untuk menemukan titik koordinat, kita perlu menghitung nilai fungsi f(x) pada x = 1, x = e, dan x = e2.
- f(1) = loge (1) = 0 karena e0 = 1.
- f(e) = loge (e) = 1 karena e1 = e.
- f(e2) = loge (e2) = 2 karena e2 = e2.
Oleh karena itu, titik koordinat yang dicari adalah (1, 0), (e, 1), dan (e2, 2).
Jawaban: Titik koordinat adalah (1, 0), (e, 1), dan (e2, 2)
