Pengertian dan Bentuk Umum Logaritma

Logaritma adalah operasi inversi atau kebalikan dari perpangkatan. Dengan begitu, logaritma mencari nilai pangkat dari sebuah bilangan.

Bentuk Umum :

Jika an = x maka alog x = n

Keterangan :

a = bilangan pokok (basis), syarat: a > 0 dan a â‰  1

x = bilangan yang dicari nilai logaritmanya (numerus), syarat: x > 0

Fungsi logaritma adalah fungsi invers dari eksponensial. Fungsi logaritma didefinisikan sebagai :

Untuk x > 0 , a > 0, dan a â‰ 1, y = log a x jika dan hanya jika x = a y
